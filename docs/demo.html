Code & Preformatted

Import the reset in your CSS:

@import "@effective/css";

/* Your layers come after */
@layer components {
  .card {
    padding: 1rem;
    border-radius: 8px;
  }
}

Why use CSS layers?

CSS @layer gives you explicit control over the cascade order. Instead of relying on source order or specificity hacks, you declare your layers upfront and the browser respects that order regardless of where styles are defined.

Media Elements

Media elements are set to display: block and max-inline-size: 100% to prevent overflow and remove the unwanted whitespace below inline images.
