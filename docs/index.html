Why @effective/css

Built for the modern stack

Layered Architecture

Uses native CSS @layer for explicit cascade control. No more specificity wars or !important hacks.

Modular by Design

Import the full reset or pick individual layers. The elements layer is completely optional.

Web Component Safe

No inheritance patterns that break Shadow DOM. Direct box-sizing ensures consistent behavior across Custom Elements.

Accessibility First

Focus indicators preserved. Respects prefers-reduced-motion. No aggressive resets that harm usability.

Evergreen Only

Targets modern browsers without legacy baggage. Smaller, cleaner, faster. No IE hacks or outdated workarounds.

Zero Dependencies

Pure CSS. No preprocessors required. No build step needed. Just import and go.
